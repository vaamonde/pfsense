Habilitando os Recursos de Load Balancing (Balanceamento de Carga) no pfSense
System
	Advanced
		Miscellaneous
			Load Balancing: ON
			Set the source tracking for sticky connections: Default
		Gateway Monitoring
			State Killing on Gateway Failure: ON
			Skip rules when gateway is down: ON
	Save

Criando o Grupo de Gateway das Interfaces WAN de Load Balancing (Balanceamento de Carga) no pfSense
System
	Routing
		Gateway Groups
			+ADD
				Edit Gateway Group Entry
					Group Name: LB_WAN
					Gateway Priority
						WANVIVO_DHCP Tier 1  Interface Address  Interface WANVIVO_DHCP
						WANNET_DHCP  Tier 1  Interface Address  Interface WANNET_DHCP
					Trigger Level: Packet Loss or High Latency
					Description: Grupo de Gateways LB das Interfaces WAN Vivo e Net
				Save
			Apply Changes

				 Hour: *
				 Day of the Month: *
				 Month of the Year: *
				 Day of the Week: *
				 User: root
				 Command: /usr/local/sbin/pfSsh.php playback svc restart dpinger
			Save
